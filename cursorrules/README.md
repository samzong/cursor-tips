## How to use

1. Install Cursor: https://cursor.com/
2. Open Cursor Settings, and enable `Include Cursor rules`
3. Find your wanted cursor rules, and click `Download`
4. Rename to `.cursorrules` in Your Local Directory
5. Enjoy!
